# Gameverse

Gameverse is a modern gaming landing page inspired by the zentry website. It shows interactive animations, video backgrounds, and responsive layouts built using react and tailwind css.

## Live Demo

[View Live Demo](https://gameverse-sigma-ashy.vercel.app/)

## technologies

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- GSAP
- React Icons

## features

- fully responsive layout
- immersive hero section with video backgrounds
- animated navigation bar
- interactive bento grid layout
- smooth tilt hover animations
- GSAP-powered scrolling animations
- reusable React components
